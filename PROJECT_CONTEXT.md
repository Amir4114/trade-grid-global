# Trade Grid Global

## Vision

Trade Grid Global is a B2B food trading platform connecting importers, exporters, manufacturers, distributors, and wholesalers.

Focus: Food & FMCG only.

Goal: Become the world's most trusted food trading platform.

## Core Modules

* Authentication
* Company Profiles
* Verification System
* Product Catalog
* RFQ Marketplace
* Admin Dashboard

## User Roles

### Buyer

* Browse suppliers
* Browse products
* Post RFQs
* Receive quotations

### Supplier

* Create company profile
* Upload products
* Upload certifications
* Respond to RFQs

### Admin

* Verify companies
* Verify certifications
* Approve products
* Moderate RFQs

## Verification Levels

* Unverified
* Pending Verification
* Verified
* Premium Verified
